#  Sistema de Biblioteca Ativa Escola

Projeto desenvolvido para o **Projeto Integrador (PI)** da Univesp. O sistema visa modernizar a gestão do acervo escolar e incentivar o hábito da leitura através de elementos de gamificação.

##  Tecnologias Utilizadas
* **Linguagem:** Python
* **Framework Web:** Flask
* **Banco de Dados:** SQLite3
* **Versionamento:** Git & GitHub

##  Foco em Gestão de Riscos
O software foi estruturado para mitigar riscos operacionais da biblioteca, como:
* Extravio de livros através do controle rigoroso de inventário.
* Inconsistência de dados utilizando um banco de dados relacional.
* Desinteresse dos alunos através de um sistema de engajamento.

##  Como Executar
1. Instale as dependências: `pip install flask`
2. Inicialize o banco de dados: `python init_db.py`
3. Rode a aplicação: `python app.py`
